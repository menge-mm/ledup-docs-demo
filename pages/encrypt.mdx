# Function: encrypt()

---

```ts
encrypt(data, key): EncryptOutput
```

Encrypts a given string using the AES-256-CCM encryption algorithm. The function generates a random initialization vector (IV), encrypts the data with the provided key, and returns the encrypted data along with the IV and authentication tag. AES-256-CCM (Counter with CBC-MAC) provides encryption, integrity, and authentication.

### Parameters

- **data**: `string`  
  The string data to be encrypted.

- **key**: `CipherKey`  
  A 32-byte encryption key required for AES-256 encryption.

### Returns

- **EncryptOutput**  
  An object containing the following:
  - `iv`: Initialization vector (IV)
  - `tag`: Authentication tag
  - `encrypted`: The encrypted data

### Throws

- Throws an error if the encryption process fails.

### Example

```ts
const data = 'Sensitive information';
const key = crypto.randomBytes(32); // Generate a secure key
const result = encrypt(data, key);
console.log(result);
// Output: { iv: "hexString", tag: "hexString", encrypted: "hexString" }
```
